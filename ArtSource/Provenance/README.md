## 使用约束

- 项目定位为非商业同人学习作品。
- 官方参考图不放入可发布构建，也不作为运行时素材。
- 后续生成的正式贴图需要记录日期、用途、版本、参考图和生成要求。
- 正式运行时资源经过切图、清理、尺寸与导入设置确认后，才复制到 `Assets/Game/Art/`。
